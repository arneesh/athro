# Linear Search

Pass the array of **numbers/string** along with the element which
needs to be found.

## Usage

```tsx title="src/sample/linearsearch.ts"
import { linearSearch } from 'athro';

const arr = [2, 6, 3, 8];
const target = 6;

const result = linearSearch(arr, target)
```
